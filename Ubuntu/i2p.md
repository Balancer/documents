# Установка i2p

	sudo apt-add-repository ppa:i2p-maintainers/i2p

(или ```sudo add-apt-repository ppa:i2p.packages/i2p```)

	sudo apt-get update && sudo apt-get install i2p

	i2prouter start

* Управление: http://127.0.0.1:7657/
